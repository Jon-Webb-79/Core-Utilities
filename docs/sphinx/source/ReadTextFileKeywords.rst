ReadTextFileKeywords
====================
This section documents the ``ReadTextFileKeywords`` class encoded in the
``read_files.py`` file.  This class reads data from a .txt file and returns
the data as a specific, user defined data type.  This class specifically
Looks for a keyword and reads the data to the right of the keyword.
